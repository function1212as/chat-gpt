# chat-gpt

``` text
基于 gpt-3.5-turbo 模型
这是一个打包好的chat-gpt应用可以点击index.html打开使用。（需要梯子）
```

# 里边内置语音转化功能，需要使用node来启动服务

```js
在当前文件夹内 命令 node bundle.js
注意node版本不低于14
```
